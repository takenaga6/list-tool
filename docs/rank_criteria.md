## 理想顧客プロファイル（Offi-Stretch 契約実績ベース）

| 軸 | 内容 |
|---|---|
| **業種** | 情報通信・金融・人材・商社・コンサル・士業等の高利益率B2B |
| **意識** | 社長 or ウェルフェア担当者が健康意識高い |
| **ブランド** | PR媒体に掲載（承認欲求・ブランド意識高い） |
| **福利厚生** | 法定外福利充実だがフィジカルケアは未着手 |
| **規模・拠点** | 自社ビル or 固定オフィス、従業員10〜100名、単一〜少拠点 |

## 採点表（各項目1点 + ボーナス）

### ボーナス点（経路依存）

| # | シグナル | 点数 | 根拠 |
|---|---|---|---|
| B1 | ★PR媒体クエリ経由（クエリ内に媒体名） | **+2** | クエリに媒体名が含まれる = 掲載確定 |
| B2 | ★健康経営優良法人リスト経由（経産省） | **+2** | kenko-keiei.jp から発見 = 認定確定 |
| B3 | ★PR媒体リストページ経由（B1と排他） | **+2** | PR媒体一覧ページから発見 = 掲載確定 |

### 通常採点（各1点）

| # | シグナル | 評価根拠 |
|---|---|---|
| 1 | PR有料媒体掲載（URL or テキストで検出） | 承認欲求・ブランド意識のシグナル |
| 2 | 健康経営メディア掲載（アクサ/大同/健康経営の広場等） | 健康意識のシグナル |
| 3 | 法定外福利厚生（マッサージ・社食・ジム等） | 社員へのケア投資実績 |
| 4 | 福利充実 + フィジカルケア未着手 | 提案余地が大きい |
| 5 | 健康経営への注力（健康経営優良法人・ウェルビーイング等） | ターゲット企業の特徴 |
| 6 | 健康セミナー登壇・社内健康推進担当あり | 健康投資継続意向 |
| 7 | 経営者の健康意識・メディア露出（社長インタビュー等） | 意思決定者へのアクセス |
| 8 | PR/広告積極投資（受賞・掲載・テレビ出演等） | 体験投資にも積極的 |
| 9 | 売上拡大・自社ビル・高利益シグナル | 予算余力あり |
| 10 | 高利益率B2B業種（IT/金融/人材/商社/コンサル/士業） | フィジカルケアが特に刺さる |
| 11 | 従業員数 20〜100名 | 契約実績の最頻出サイズ |
| 12 | 単一拠点（支店・支社・営業所なし） | 固定オフィス訪問可 |

## 自動登録・スキップ閾値

| 条件 | 動作 |
|---|---|
| スコア **8点以上** かつ 信頼度 **2以上** | 確認モードでも自動登録（HubSpot） |
| スコア **3点未満** | 候補リストにも出さず自動スキップ |

（閾値は `config.py` の `AUTO_REGISTER_SCORE / MIN_PENDING_SCORE` で変更可）

## 精度改善のポイント

- **フィジカルケア未着手（項目4）が特に重要**: 食事補助・社食・ジム等はあるが、マッサージ・ストレッチ系がない企業 → 提案余地最大
- **経産省健康経営優良法人リスト（B2）**: 認定確定 = スコア+2は妥当、さらに健康経営注力（項目5）とも重複してカウントされやすい
- **PR媒体経由（B1/B3）**: 掲載費を払っている = 社長の承認欲求・ブランド意識 → 出張施術のような「見せ方のある福利厚生」が刺さりやすい
